# Apps Directory

This directory contains the user-facing and runner applications of the personal finance automation platform.

## Applications

* **[web](./web/)**: The React frontend web application, serving as the user interface for transaction review, manual entry, dashboard reports, and platform settings.
* **[api](./api/)**: The Node.js API server backend, serving the business logic and orchestrating between database stores, automation candidates, and Hledger ledger interfaces.
